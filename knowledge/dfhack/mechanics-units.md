# Mechanics Units

## VERIFIED

Commands for dwarf unit manipulation:
- `dwarf-op`: Tune units to perform underrepresented job roles in your fortress.
- `dwarfvet`: Allow animals to be treated at hospitals.
- `fastdwarf`: Citizens walk fast and finish jobs instantly.
- `startdwarf`: Change the number of dwarves you embark with.
- `superdwarf`: Make a dwarf supernaturally speedy.

## OPEN

`help unit` returns no entry, indicating potential undocumented API behavior or tagging misclassification.
